  <p>This example shows the DataGrid component with auto wrap column cell content. In this example, you can see that the <strong>Main Fields of Invention</strong> column cell content exceeded the available width, so it was wrapped into multiple lines. </p>

<ActionDescription>
   <p><a target='_blank' href='https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.SfGrid-1.html#Syncfusion_Blazor_Grids_SfGrid_1_AllowTextWrap'>AllowTextWrap</a> is a property that enables the Blazor DataGrid Autowrap cell content. When the width of the content and header cell content exceeds the width of the cell, this property will wrap the cell text in multiple lines.</p>
   <p>The <code>AllowTextWrap</code> property is enabled in this demo, as seen in the <strong>Main Fields of Invention</strong> column, where the content is wrapped into multiple lines because it exceeds the cell width.</p>
   <p>More information about the Autowrap feature can be found in this <a target='_blank' href='https://blazor.syncfusion.com/documentation/datagrid/cell/#auto-wrap'>documentation</a> section.</p>
</ActionDescription>
