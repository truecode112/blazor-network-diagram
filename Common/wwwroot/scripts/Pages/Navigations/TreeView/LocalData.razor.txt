<ActionDescription>
   <p>The <code>TreeView</code> component loads the data through the <code>DataSource</code> property, where the data can be either local data or remote data. In case of local data, the data structure can be hierarchical data or list data (with self-referential format i.e., mapped with the <code>Id</code> and <code>ParentID</code> fields). In this demo, the first <code>TreeView</code> is bound with the hierarchical data that contains array of nested objects. And the second <code>TreeView</code> is bound with the list type data where the parent-child relation is referred by the <code>Id</code> and <code>ParentID</code> mapping fields. More information about <code>TreeView</code> can be found in this <a target='blank' href='https://ej2.syncfusion.com/aspnet-core-blazor/documentation/treeview/data-binding/'>documentation</a> section.</p>
</ActionDescription>
